FAQ

Does Wallpaper Engine work after closing the window?

Yes. When background mode is enabled, the app keeps running from the tray and continues changing wallpapers until you quit it from the tray or terminate the process.

Where are wallpapers stored?

Downloaded wallpapers are stored in the app cache on your local machine and can be deleted from the Library screen or cleared all at once.

Do I need API keys?

Some providers work best with API keys. Wallhaven SFW search and fallback placeholders can still work without provider credentials.
